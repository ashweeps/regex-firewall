# regex-firewall

Name: Ashley Espinoza  
Course: 2nd Semester  
Section: A  

This repository contains my solution for the lab "Regular Expressions on a Firewall Log."

## Tasks

- Task 1: Count firewall events and ignore header lines.
- Task 2: Count DROP or REJECT traffic.
- Task 3: Count events whose source IP address begins with 11.
- Task 4: Count events with a 7-digit size field.
- Task 5: Reformat event lines using sed.
- Task 6: Count ACCEPT TCP events with destination port 80.
- Task 7: Count events between 00:00:00 and 02:59:59.
- Bonus: Match only the valid host names with one regex.
